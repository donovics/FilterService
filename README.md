# Taboola R&D Intern Homework — Backend Track

This project is the solution for the given homework.

## It contains:

.git    # Files required for git.\
.idea   # Intelij idea setup files (not required).\
.mvn    # Files required for Maven.\
agent   # Folder containing the AI agent's .md file.\
data    # The copy of the provided data folder.\
output  # Folder containing the output files.\
src     # Folder containing da source files.\
target  # Folder containing the .jar file.\
promts/ # Folder containing the used promts.\
.gitignore  # Git assist file.\
pom.xml # Maven's pom file.\
REFLECTION.md   # Description of experience.\

## Running the service:

Navigate to the project folder and start:
java -jar target/FilterService-1.0-SNAPSHOT.jar

To clean the output folder, the service has to be restarted.
